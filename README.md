## 概要
組み込み用途向けにお使いいただける軽量なCLI(Command Line Interface)機能をライブラリとして提供します。  

CLIライブラリが要求する動作環境として、以下を満たす必要があります。  
- 対象のマイコンに通信IFが備わっている、または拡張等により通信IFが利用可能になること
- 通信IFを介して、Tera Term等のターミナルソフトが利用できること
